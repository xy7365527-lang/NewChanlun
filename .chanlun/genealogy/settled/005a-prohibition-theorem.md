## 定理陈述

**在完全分类体系中，不允许非对象来源否定对象。**

- 完全分类意味着：每一段市场运动都恰好属于一个走势类型，分类之外没有剩余
- 同级别分解意味着：走势类型连续拼接，无间隙，前一个结束就是后一个开始
- 逻辑蕴含：如果分类之外没有剩余，那么分类之外的东西（超时、阈值、非对象规则）不能参与分类之内的操作（否定走势对象）

## 推导链

```
前提1: 完全分类 [旧缠论] — 任何走势都属于且仅属于一种走势类型
前提2: 同级别分解 [旧缠论] — 走势类型连续拼接，无间隙
────────────────────────────────────────────────────────────
结论: 分类之外的存在者不能否定分类之内的对象
      （否则分类不完全——存在分类之外可以影响分类之内的东西）
```

## 编码约束（定理直接推论）

1. **禁止**在任何对象生命周期中引入超时（timeout）作为否定条件
2. **禁止**使用阈值（threshold）来决定对象是否有效
3. **禁止**在对象之外设置"全局规则"来裁决对象的存亡

## 与 005 命名半的关系

本定理是 005（对象否定对象）原则的**必要条件但不是充分条件**。

- 本定理只说"不能做什么"（否定性）
- 005 命名半还说"应该做什么"（肯定性：对象的否定必须来自对象事件）
- 你可以接受本定理而不接受命名半（只禁止超时，但不要求正面寻找对象事件）
- 你不可以接受命名半而不接受本定理（宣称对象否定对象，但允许超时）
