## 语法规则陈述

**对象否定对象**：在本体系中，一个对象被否定的唯一来源是：
1. **内在否定**：对象自身产生的否定（如中枢被自身的突破段否定）
2. **外部对象生成**：另一个同级或更高级别对象的生成否定了当前对象

体系中不存在第三种否定来源。

## 语法规则的性质

这不是一个经验命题。它不是在说"市场中的走势事实上以对象否定对象的方式运动"。

它是**走势描述的语法规则**：它规定了什么算合法的走势转换描述。

- "这个 anchor 因为超时所以消失了" → **语法不合法**（不是"为假"，是"不成立"）
- "这个中枢因为被突破段生成所以结束了" → **语法合法**
- "这个走势因为统计阈值触发所以完成了" → **语法不合法**
- "这个走势因为背驰（内在否定）所以完成了" → **语法合法**

类比：国际象棋中"这盘棋因为下了三十分钟所以结束了"不是一个为假的棋局描述，是一个语法上不合法的棋局描述。棋局结束只能因为将杀、认输或和棋协议。

## 与定理半（005a）的关系

005a（禁止性定理）是本规则的**必要条件但不是充分条件**。

- 005a 只说"不能用超时/阈值"（否定性）
- 本规则还说"必须找到对象事件作为否定来源"（肯定性）
- 这个正面内容**不可从完全分类和同级别分解推出**——它是编排者的原创命名

## 哲学定位

### 谢林的开端
缠论给出了完全分类、同级别分解、走势必完美等定理。这些定理背后有一个共同结构。编排者将这个结构命名为"对象否定对象"。这个命名行为不可从缠论内部推导——缠论给了结构，但没给名字，也没给"应该把它独立为原则"的判断。这是一个**谢林式的原初命名**：从定理到命名的跳跃不可由系统完成。

### 维特根斯坦的语法
"对象否定对象"是缠论这套走势描述语言的**语法规则**（Grammatik）。语法规定什么是有意义的走势描述、什么是无意义的。语法在真假之前——一个语法不合法的描述不需要被检验就可以被排除。

### 维特根斯坦的铰链命题
作为语法规则，本原则是系统质询的**条件**而非**对象**。Agents 不质询它——agents 用它来质询其他东西。唯一能动摇语法规则的是实践的持续崩溃（系统性、结构性、长期的负结果），而这个判断不能从语法内部做出，只能由编排者做出。

### 意义的生产方式
本规则的意义在确立时刻是稀薄的。它的后续使用历史才是其意义的载体（维特根斯坦：Bedeutung ist Gebrauch）。第一次质询循环赋予它第一层意义，第一次矛盾赋予它边界，第一次概念分离改变它的范围。

## 交易理性的语法基础

合法交易 vs 非法交易的区别不是盈亏的区别：
- 非法交易（用超时判断 dead anchor 消失后入场）可以碰巧赚钱——但这笔钱来自运气，不来自语法，运气不可重复
- 合法交易（严格遵循定义的走势判断后入场）可以亏钱——但亏损属于走势发展的合法可能性之内
- **可重复性来自语法的合法性，不来自单次结果的盈亏**

## 边界条件

- 如果某个对象永远不被否定（如最后一个未闭合中枢），它保持其状态直到新数据到来产生新对象
- "不被否定"不等于"被确认"——未闭合的对象保持 candidate 状态
- 级别递归中，高级别对象否定低级别对象符合此规则——高级别对象也是对象
- 语法不能自我奠基：本规则的合法性不能从语法内部证明，只能通过长期交易实践（生活形式的最终判据）检验
