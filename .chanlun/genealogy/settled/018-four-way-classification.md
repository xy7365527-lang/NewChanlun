# 018 — 四分法扩展：行动类（l'acte）

**状态**: 已结算
**日期**: 2026-02-17
**来源**: maimai v1.0 发布事件。该事件中 escalation 不携带信息差，编排者审查环节为空，暴露了原三分法的盲区。

## 分离

原三分法（定理 / 选择 / 语法记录）无法分类以下产出：
- 不推导新知（非定理）
- 不在多条路中选（非选择）
- 不把隐性规则显式化（非语法记录）
- 纯粹是操作性事件（版本升级、状态迁移、发布）

## 新定义

**行动类（l'acte）**：不携带信息差的操作性事件 → 自动执行，无需编排者审查。

四分法：
1. **定理** → 自动结算
2. **选择** → 编排者决断
3. **语法记录** → 编排者辨认
4. **行动** → 自动执行
