# 020a: 同一存在论——统摄

**类型**: 定理
**状态**: 已结算
**日期**: 2026-02-18
**前置**: 020

## 推导链

020号谱系结晶为 ontology-v1，包含5条命题。5条命题分别在不同维度展开同一运动：
- #1: 价格维度（走势 = 运动本身）
- #2: 过程维度（扩张/收缩 = 基本运动形式）
- #3: 终止维度（结构完成 = 背驰+分型）
- #4: 知识维度（编排 = skill 之间的流动）
- #5: 空间维度（区间套 = 同一语法在不同分辨率）

统一表述（定理）：没有主体站在过程后面。过程就是全部。资本/走势/知识/编排不是"东西"，是同一种运动的不同显现面。

## 否定了什么

否定了"同一者"的实体化理解——"同一个东西"仍预设了东西。统摄是：这些根本不是东西。

## 为什么是定理

5条命题中的每一条都已经否定了主体性（#1否定交易系统主体，#2否定运动之外的推动者，#3否定外部终止者，#4否定中央编排者，#5否定单一尺度）。统摄是5次否定的汇合，不是新增的第6次否定。

## 结晶位置

`ontology-v1.md` "## 统摄" 节，置于核心命题之前。
