# 025 等价关系降格：从本体关系到筛选条件 → 流转关系诞生

- **状态**: 已结算
- **日期**: 2026-02-18
- **结算日期**: 2026-02-18

- **结算方式**: 编排者决断 → 新定义 #14（liuzhuan.md）

## 矛盾/发现

编排者指出：**缠师自己没有等价关系**。缠师的操作方式是三独立系统——A 系统、B 系统、A/B 系统各自跑缠论，各自完备。不需要先判断"等价"才能看比价。

这意味着 C-1/C-2/C-3 **不是等价关系的判定条件**，而是**"哪些比价系统值得看"的筛选条件**。性质完全不同：

- 等价关系：预设两个对象之间存在某种本体论上的关系
- 筛选条件：只是说"这个比价系统的走势结构是否有交易价值"

## 编排者决断

### 关键属性（编排者确认）

1. **有方向**：A/B 上升 = 资本从 B→A。流转是有向的。
2. **一对多**：一个角的收缩可能对应三条比价线（该角与其他三角各一条）。
3. **共振涌现**：多条比价走势同方向运动 = 资本从一个角同时向多个角流出。

### 决断

- 流转关系是**新定义**（#14, `liuzhuan.md`），**不是**等价关系（#13）的 v2
- 理由：等价是 pair-level 筛选（L1），流转是 topology-level 涌现（L2）。混在一起是语义污染。
- 形式化简洁程度本身就是论证——方向、聚合、共振，三步完。

### 编排者补充

- **守恒约束**不只是数学性质，是"资本不凭空产生也不凭空消失"的形式表达。守恒破缺本身是信号（跨区域流动）。
- **结构性同步**而非时钟性同步——不同边的走势各自有级别和节奏，共振是结构上的同步。与缠论级别自相似性一致。
- **强共振（|net|=3）vs 弱共振（|net|=2）**可能对应趋势与盘整——后续验证，当前不急。

## 推导链

1. 缠师用三独立系统操作，无"等价"概念 [旧缠论]
2. C-1/C-2/C-3 在缠师框架下是筛选条件（哪些比价系统值得跑），不是本体关系
3. L2（全球资本流转形式化）需要知道哪些比价系统在描述同一股资本运动
4. 编排者指出：这不是经典等价关系——一对多、有方向、共振涌现
5. → L1 层面：C-1/C-2/C-3 是筛选条件，保留"等价"名称但明确其筛选性质
6. → L2 层面：新定义"流转关系"（#14）——四矩阵上的有向流场
7. 流转关系 = 对每个顶点 V，聚合其 3 条边的 StrokeFlow 方向，≥2 条同向 = 共振
8. 4 个顶点 net flow 之和 = 0（守恒约束）
9. 编排者确认：方向对，做。
