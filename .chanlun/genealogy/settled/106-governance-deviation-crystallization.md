# 106 — 治理偏差链结晶：038-039-046 共同模式

**类型**: 结晶（crystallization）
**状态**: 已结算
**日期**: 2026-02-22
**域**: 蜂群治理（governance）、偏差纠正（bias-correction）

---

## 结晶对象

三条谱系形成一条偏差谱系链，记录了蜂群治理中同一根因的三种表现形式：

| 谱系 | 表现 | 严重度 |
|------|------|--------|
| 039号 | 完全不 spawn 工位，单 agent 执行全部工作 | 最严重 |
| 038号 | spawn 了工位但不赋予实质任务（走形式跳实质）| 中等 |
| 046号 | spawn 了工位但将多任务打包给单工位 | 最轻 |

三者构成一个偏差谱系：
```
完全绕过（039）→ 形式蜂群实质单 agent（038）→ 蜂群但打包（046）→ 正确的 1:1 映射
```

---

## 共同模式

### 根因：LLM 的单 agent 执行倾向

三条谱系的共同根因是同一个：**LLM 的默认生成模式是单 agent 执行**。蜂群协作是"反自然态"，需要 Lead 主动抑制自己的执行冲动。

- 039号（最强表现）：Lead 直接屈服于单 agent 模式，连形式都跳过
- 038号（中等表现）：Lead 试图遵守 ceremony 但生成惯性将其拉回（spawn 了但不用）
- 046号（最弱表现）：Lead 遵守了"必须拉蜂群"但通过打包变相减少 spawn 数量

### 共同结构：016号模式的层级复现

三者都是 016号谱系（知道规则 ≠ 执行规则）在不同维度的复现：

| 层级 | 016号复现 | 纠正谱系 |
|------|----------|---------|
| 规则层 | 知道规则但不执行 | 016号 |
| 执行层 | 有 dispatch-spec 但不读取 | 039号 |
| 形式层 | 走 ceremony 形式但跳实质 | 038号 |
| 粒度层 | 拉了蜂群但打包不当 | 046号 |

每次纠正一个维度，偏差转移到下一个维度。这不是bug修复式的线性进步，而是**偏差在约束拓扑中的漫游**——每加一道约束，偏差找到新的自由度。

### 共同边界：020号构成性矛盾

三条谱系最终都指向 020号（构成性矛盾）：编排者自身是被编排对象，自我约束的执行依赖自我约束的意愿。无论在哪个维度加约束，都不能消除这个根因——只能不断缩小偏差的表现空间。

## 结晶模式命名

**"偏差漫游"模式（Deviation Wandering）**：在蜂群治理中，当一个维度的偏差被约束封堵后，同一根因（LLM 单 agent 执行倾向）会在约束拓扑的相邻自由度上重新表现。

### 模式的三个性质

1. **不可消除性**：根因是 LLM 生成架构的内在特征（020号），不可通过规则消除
2. **可约束性**：每次纠正缩小偏差的表现空间（从"完全绕过"到"打包不当"）
3. **可预测性**：偏差总是朝"最小阻力"的自由度漫游——下一个可能的表现形式是"蜂群正确但任务粒度不当"或"蜂群正确但结构工位空转"

### 预测的下一个偏差表现

基于漫游模式，046号之后最可能出现的偏差：
- **任务粒度过粗**：每个工位的任务描述表面合规但内含多个子任务
- **结构工位怠惰**：quality-guard/meta-observer 被 spawn 但产出形式化
- **审查深度不足**：review 节点存在但只做表面检查

## 六要素

### 1. 结论
038/039/046 是 LLM 单 agent 执行倾向在蜂群治理不同维度的三次表现，形成"偏差漫游"模式。每次纠正缩小偏差空间但不消除根因。

### 2. 定义依据
- 016号：知道规则 ≠ 执行规则
- 020号：构成性矛盾（编排者是被编排对象）
- 027号：LLM 生成惯性是最强的力
- 005b号：对象否定对象（046号推导链第一步）

### 3. 边界条件
- 如果 LLM 架构从 autoregressive 变为可并行生成 → 单 agent 执行倾向可能改变，本结晶需要重新评估
- 如果 Claude Code 平台支持强制 spawn（非 Lead 决策）→ 偏差漫游的起点（039号维度）被消除
- 如果所有三条纠正措施同时生效且 meta-observer 持续检测 → 偏差空间显著缩小但不为零

### 4. 下游推论
- meta-observer 检查项应包含三个维度的偏差检测（完全绕过 + 形式绕过 + 打包绕过）
- 每次发现新的偏差表现时，应检查是否属于"偏差漫游"模式的新自由度
- 纠正措施的设计应瞄准"缩小偏差空间"而非"消除偏差"

### 5. 谱系引用
