# meta-orchestration

蜂群元编排方法论 — AI Agent蜂群的组织原则。

## 这是什么

资本运动形式语法的自我运行系统。

核心观点：

- **新缠论 = 资本运动的形式语法**。笔-线段-中枢-走势是扩张/收缩辩证运动在价格维度的显现形式。这套语法同时适用于系统自身的运行。
- **蜂群是矛盾显现机器**，不是代码生产线。形式化过程中暴露的概念矛盾是最有价值的产出。
- **绕过矛盾是最大的敌人**。Workaround扼杀新知识的诞生。
- **编排者的位置是欲望和决断**，不是控制。AI有生成性但没有方向，编排者有方向但不需要懂代码。
- **谱系是系统的免疫机制**。记录每一次矛盾、分离、否定，防止系统退化回已被否定的方案。
- **元编排是 skill 集合**，不是中央控制器。编排能力在 skill 网络中流动，没有"最终编排者"凌驾于系统之上。

## 适用场景

- 将领域知识形式化为代码的项目（如量化交易系统、法律规则引擎等）
- 编排者是领域专家而非程序员
- 项目中概念定义本身在演化
- 需要多agent协作并保持概念一致性

- superpowers管工程层（TDD、代码审查、调试）
- 本skill管概念层（质询循环、矛盾处理、谱系）
- 当两者冲突时，概念层优先

## 理论谱系

本方法论的生成过程：
1. 从Nowak《超级合作者》和Chwe《理性的仪式》出发
2. 否定了核心预设（AI无背叛动机，人类合作理论不适用）
3. 保留并重新定义了拓扑、共同知识、图书馆/广场区分
4. 从蜂群本体论重新推导：接收→产出→传递
5. 产出质询循环、欲望驱动、生成态/结算态、回溯结算、谱系免疫
6. **020号存在论位移**：发现扩张/收缩是所有层次的基本运动形式——价格、比价、系统运行、上下文工程。系统从"用缠论做交易"变为"用资本运动形式语法运行自身"
7. 分布式编排：元编排本身 = skill 集合，自指困境被拓扑地消解
