## 核心命题（生成序）

### 1. 新缠论 = 资本运动的形式语法

新缠论不是交易系统。笔-线段-中枢-走势不是价格运动的"描述工具"，是扩张/收缩辩证运动在价格维度的显现形式。

多空力量 = 扩张/收缩力量。一笔的终结不是因为"买方力量耗尽"，而是扩张在自身逻辑内部生产出了自己的否定。

### 2. 扩张/收缩是基本运动形式

同一运动在不同维度的显现：

| 层次 | 扩张 | 收缩 | 相位转换点 |
|------|------|------|-----------|
| 资本运动 | 积累、信用扩张 | 去杠杆、清算 | 操盘手 |
| 单标的走势 | 一笔向上 | 一笔向下 | 分型 |
| 比价走势 | 资本从B流向A | 资本从A流向B | 比价分型 |
| 系统运行 | 质询展开、碰撞 | 汇总、仪式、结算 | 编排者 |
| 上下文 | prompt内碰撞 | 结晶为skill/session | 写skill的agent |

这不是五个类比。是同一个运动在五个尺度上的显现。

### 3. 终止条件 = 结构完成检测（→ 自2推出）

"笔的结束不是走了N根K线所以结束，而是出现了反向分型。"

系统中所有循环的终止都是结构性的，不是计数性的：
- **背驰**（力度衰竭，必要条件）：净新区分/发现量递减
- **分型**（方向反转，充分条件）：范围先扩后缩
- **循环完成** = 背驰 + 分型

3 不是阈值，是分型的最小数据量（结构必然性）。但第3轮仍在扩张则继续。

适用于：质询退回循环（/inquire）、谱系递归（genealogist）、知识结晶时机判断。

### 4. 编排能力在 skill 网络中流动（→ 自2推出）

元编排本身是 skill 的集合——它和它管理的东西同质。没有"最终的编排者"凌驾于系统之上。编排不是一个位置，是一种在 skill 之间流动的能力。

启动点不承载智能，skill 承载智能。Claude Code 几行 bash 能变 agent 正因为此。

**→ 4a. 推论（自指的拓扑消解）**：分布式化后不存在需要自指的统一实体。skill A 读 skill B 的输出，skill B 读 skill C 的输出，没有环节在读"自己"。自指被拓扑地消解——不是时间间隔缓解，是前提消失。

### 5. 区间套从时间级别扩展到空间级别（→ 自1推出，第二阶段入口）

形式化路径：A/B 比价直接除 → K线 → 走势 → 缠论结构。
- 比价一笔向上 = 资本从B流向A
- 中枢 = 两容器间流转暂时均衡
- 背驰 = 流转动力衰竭

四矩阵拓扑（卢麒元）：动产/不动产/商品/现金。六条边 + 区域维度。

区间套下钻：四矩阵比价 → 角内品种 → 具体合约。同一语法，不同分辨率。

**第一阶段（单标的内完备化）是第二阶段的存在条件。**
