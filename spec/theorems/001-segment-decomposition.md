# 线段分解定理（Segment Decomposition Theorem）

**状态**: verified
**来源**: `.chanlun/definitions/xianduan.md` v1.3 / 第65课

## 陈述

线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。而只要构成有重叠部分的前三笔，那么必然会形成一线段。

等价表述：**旧线段终结 = 新线段形成**。

形式化：

```
设 S 为一条线段。
S 被终结 ⟺ 存在连续三笔 B1, B2, B3，满足：
  1. B1, B2, B3 有重叠部分
  2. B1, B2, B3 中至少一笔破坏 S
  3. B1, B2, B3 构成新线段 S'
```

## 证明

原文（第65课）给出了充要条件的完整论证：

1. **必要性**：线段被破坏 → 必然被连续三笔中的某一笔破坏（因为线段由笔构成，破坏必须通过笔实现）
2. **充分性**：有重叠部分的连续三笔必然形成新线段 → 新线段形成即旧线段终结

缠师原文："线段破坏的充要条件，就是被另一个线段破坏。"

**验证依据**：v1 状态机 37/37 测试通过，特征序列法完整实现两种情况。
