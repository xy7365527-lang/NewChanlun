# 精确大转折点寻找程序定理（区间套 / Interval Nesting Theorem）

**状态**: verified
**来源**: `.chanlun/definitions/beichi.md` v1.1 / 第27课

## 陈述

某大级别的转折点，可以通过不同级别背驰段的逐级收缩范围而确定。

形式化：

```
给定 L_n 级别背驰段 D_n，
在 D_n 内部用 L_{n-1} 级别找 D_{n-1}，
...
直到最低级别 L_1。

结果：D_n ⊃ D_{n-1} ⊃ ... ⊃ D_1
转折点 ∈ D_1
```

嵌套条件：
1. **范围收缩**：低级别背驰段时间范围 ⊂ 高级别背驰段
2. **级别递减**：每次递推必须降级
3. **背驰独立成立**：每一级别的背驰在该级别走势中独立满足

终止条件：
1. 到达最低可观察级别（通常1分钟）
2. 最低级别背驰段完成（转折出现）
3. 力度否定：某级别走势力度超过前段 → 该级别区间套假设被否定

## 证明

理论依据：低级别背驰是高级别背驰的**必要条件**（非充分条件）。

原文（第27课 L42-46）："学过数学分析的，都应该对区间套定理有印象。这种从大级别往下精确找大级别买点的方法，和区间套是一个道理。"

数学类比：闭区间嵌套定理。但级别有限，不能精确到一点——收缩到最低级别的背驰段即为最精确定位。

**验证依据**：`nested_divergence_search()` 20测试全GREEN，跨级别搜索实现完整。
