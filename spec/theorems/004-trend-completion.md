# 走势终完美原理（Trend Completion Axiom）

**状态**: verified
**来源**: `.chanlun/definitions/zoushi.md` v1.6 / 第17课

## 陈述

缠中说禅技术分析基本原理一：任何级别的任何走势类型终要完成。

形式化：

```
∀ level k, ∀ trend_type T at level k:
  ∃ time t: T is completed at t
```

## 证明

这是缠论体系的**公理**（基本原理），不由其他定理推导，而是从市场运行的基本事实出发：

1. 市场不可能永远单向运动（资金有限性）
2. 任何走势类型（盘整或趋势）都有完成的条件：
   - 盘整：3个连续次级别走势重叠形成中枢后即可完成
   - 趋势：2个依次同向中枢形成后即可完成
3. 完成后可以延伸，但延伸本身也终将完成

原文（第17课）："缠中说禅技术分析基本原理一：任何级别的任何走势类型终要完成。"

**公理地位**：本原理是买卖点安全性保证的基础——第二类买点的安全性直接依赖"走势终完美"。
