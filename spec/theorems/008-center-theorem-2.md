# 中心定理二（Center Theorem II / Two-Center Relation）

**状态**: verified
**来源**: `.chanlun/definitions/zhongshu.md` v1.3 / 第20课

## 陈述

前后同级别的两个缠中说禅走势中枢，后 GG < 前 DD，等价于下跌及其延续；后 DD > 前 GG 等价于上涨及其延续。后 ZG < 前 ZD 且后 GG >= 前 DD，或后 ZD > 前 ZG 且后 DD <= 前 GG，则等价于形成高级别的走势中枢。

形式化（完全三分类）：

```
设前中枢 C1，后中枢 C2。

上涨延续 ⟺ C2.DD > C1.GG
下跌延续 ⟺ C2.GG < C1.DD
级别扩张 ⟺ (C2.ZG < C1.ZD ∧ C2.GG ≥ C1.DD) ∨
             (C2.ZD > C1.ZG ∧ C2.DD ≤ C1.GG)

上涨延续 ∪ 下跌延续 ∪ 级别扩张 = 全集（完备三分类）
```

## 证明

原文（第20课）使用三次充要条件表述："等价于"、"必然要采取第三种方式"、"当且仅当"。

完备性论证：
1. 两中枢波动区间 [DD, GG] 的关系只有三种可能：完全分离（上或下）、部分重叠
2. 完全分离向上 = 上涨延续
3. 完全分离向下 = 下跌延续
4. 部分重叠 = 级别扩张
5. 三种情况穷尽所有可能 → 完备
