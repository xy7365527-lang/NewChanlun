# 禁止非对象来源否定定理（Prohibition Theorem）

**状态**: verified
**来源**: `.chanlun/genealogy/settled/005a-prohibition-theorem.md` / 第17-21课推导
**溯源**: [旧缠论:隐含]

## 陈述

在完全分类体系中，不允许非对象来源否定对象。

形式化：

```
前提1: 完全分类 — ∀ market_movement M: M ∈ exactly one trend_type
前提2: 同级别分解 — trend_types 连续拼接，无间隙
────────────────────────────────────────────────────
结论: ¬∃ non-object X: X can negate an object O within the classification
```

编码约束（定理直接推论）：
1. 禁止在任何对象生命周期中引入超时（timeout）作为否定条件
2. 禁止使用阈值（threshold）来决定对象是否有效
3. 禁止在对象之外设置"全局规则"来裁决对象的存亡

## 证明

```
1. 完全分类 → 分类之外没有剩余
2. 同级别分解 → 走势类型连续拼接，前一个结束就是后一个开始
3. 若允许分类之外的东西（超时、阈值）否定分类之内的对象
   → 存在分类之外可以影响分类之内的存在者
   → 分类不完全（矛盾）
4. ∴ 分类之外的存在者不能否定分类之内的对象 □
```

**与 005b 的关系**：本定理是否定性的（"不能做什么"）。005b（对象否定对象语法规则）是肯定性的（"应该做什么"）。本定理是 005b 的必要条件但非充分条件。
